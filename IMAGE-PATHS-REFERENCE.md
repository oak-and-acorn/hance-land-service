## How to Use in Decap CMS

1. Open the CMS at `/admin`
2. Click on any content item to edit
3. For image fields, you'll see a text input
4. Copy the path from this reference file
5. Paste it into the image field
6. Save your changes

**Note:** All paths start with `./assets/images/` for relative paths.
